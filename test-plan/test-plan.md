# Test Plan - DemoQA Website

## Objective
Validate the main functionalities of the DemoQA website through manual testing.

## Scope
Website: https://demoqa.com

Modules to be tested:
- Forms (Practice Form)
- Elements (Text Box, Buttons)

## Test Types
- Functional Testing
- Negative Testing
- UI Testing

## Tools
- Jira (Bug tracking)
- GitHub (Documentation)
- Browser (Chrome)

## Entry Criteria
- Website is accessible
- Test cases are created

## Exit Criteria
- All test cases executed
- Bugs reported
