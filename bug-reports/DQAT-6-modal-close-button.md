# DQAT-6 - Confirmation Modal Close Button Does Not Work

## Description
Close button in confirmation modal does not close the modal.

## Steps to Reproduce
1. Fill all mandatory fields with valid data
2. Click Submit
3. Wait for confirmation modal
4. Click Close button

## Expected Result
Modal should close and user should return to form page

## Actual Result
Modal remains open and does not close when clicking Close button

## Severity
High
## Priority
High

## Environment
- Browser: Chrome
- URL: https://demoqa.com
